Port of Hormuz

A real-time ship-triage game for the web

Play it on itch.io

You run the gate of a busy shipping strait. Ships arrive in waves and you sort each one fast: clear it through, pull it in to inspect, or divert it away. Clear a clean ship and it pays into your treasury; wave a dirty one through, or stop traffic you shouldn't have, and it costs a life. Three lives and the run ends. It gets quicker and dirtier the longer you last.

Status: Live on itch.io — playable in any browser

Tech Stack: React 19, Vite, TypeScript, HTML5 Canvas
